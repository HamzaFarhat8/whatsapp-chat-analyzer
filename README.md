# WhatsApp Chat Analyzer

A data analysis web application that processes WhatsApp chat data and provides meaningful insights such as message statistics, user activity, and visualizations using Python and Streamlit.

---

## Features

* Total messages, words, media, and links count
* Most active users analysis
* Monthly and daily activity timeline
* Most active time (day & hour)
* Emoji usage analysis
* Data visualization using graphs

---

## Tech Stack

* Python
* Pandas
* Matplotlib
* Streamlit

## How to Run

1. Clone the repository:

```
git clone https://github.com/HamzaFarhat8/whatsapp-chat-analyzer.git
```

2. Navigate to project folder:

```
cd whatsapp-chat-analyzer
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the application:

```
streamlit run app.py
```

---

## How to Use

1. Export your WhatsApp chat as a `.txt` file
2. Upload the file in the application
3. View insights and visualizations

---

## Notes

* Chat file is not included due to privacy reasons
* Ensure the exported chat is in `.txt` format
